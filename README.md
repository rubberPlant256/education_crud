# Система организации дошкольного образования

Стек технологий: Java 21, Spring Boot, Hibernate, liquibase, PostgreSql, Thymeleaf

Учебный проект, реализован в рамках курсовой работы.
Полная система будет поддерживать следующие роли и функции:
1.	Пользователи с ролью «Родитель» имеют возможность просматривать посещаемость и успеваемость своего (своих) ребёнка (детей).
2.	Пользователи с ролью «Преподаватель» могут:
    a.	Просматривать расписание
    b.	Просматривать и редактировать ведомости своих групп
3.	Пользователи со статусом «Администратор» могут:
    a.	Регистрировать новых учеников
    b.	Удалять учеников, которые прекратили занятия
    c.	Добавлять учеников в группы
    d.	Добавлять и удалять направления кружков
    e.	Составлять и корректировать расписание занятий

Реализована часть системы для преподавателя и родителя.
